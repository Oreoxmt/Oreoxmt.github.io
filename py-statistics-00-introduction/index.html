从零开始，跟我一起看 Coursera 呀

Start with Me | Coursera - Understanding and Visualizing Data with Python week 0

Start with Me 系列又来啦，这次是跟我一起学习 Coursera 的课程呀， 最近因为疫情的原因，密歇根大学与 Coursera 联手向中国学生提供订阅课程的优惠活动，每月仅需￥7。目前有三项最受欢迎的 Python 专项课程，分别是

  • Python for Everybody

  • Python 3 Programming

  • Applied Data Science with Python

其实每个专项课程都是可以在一个月内学完的，也就是说，只要七块钱，就可以假装在密歇根大学上课，而且还能收获一沓 University of Michigan 的课程证书哦。

我一共花了十四块钱订阅了 Python 3 Programming 和 Applied Data Science with Python 。

目前我已经完成了 Python 3 Programming 五个课程的学习，这个课程相对来说比较基础，适合初学者，容易上手，除了最后一个 Python Project: pillow , tesseract , and OpenCv 让人头大，其他的内容都可以很顺利地学下去，很推荐 👍

Applied Data Science with Python 这个课程需要有一定 Python 基础，可能不太适合我这种永远在入门的同学，主要是因为我太拖延了，所以还在学 week1 ， 但还是很推荐给大家 👍

关于这两个专项课程的学习分享可以等一等奥利奥的更新，说不定就能等到呢。

每天都在开新坑的奥利奥 🍪

最近申请了密歇根大学的专项课程 Statistics with Python 的助学金计划，等 15 天就可以免费学习啦，包括三个系列课程，分别是

  • Understanding and Visualizing Data with Python

  • Inferential Statistical Analysis with Python

  • Fitting Statistical Models to Data with Python

这个专项课程内容上循序渐进，可以让学习者对统计学有一个全面的认识，并使用 Python 来实践，又是一个入门的课程，我怎么一直在入门。

那今天先开始第一个课程啦 ~ Understanding and Visualizing Data with Python ， 看起来应该是关于数据可视化嘛
